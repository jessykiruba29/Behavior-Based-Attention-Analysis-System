# 🧠 Behavior-Based Attention Analysis System

A real-time AI-powered system that analyzes user attention levels using computer vision and behavioral cues from live video streams.

This project detects facial presence, head movement, gaze direction, blink rate, mouse activity, and browser behavior to generate dynamic attention scores and session analytics.

---

## 🚀 Features

-  Real-time webcam monitoring  
-  Face detection using OpenCV  
-  Blink rate detection  
-  Head orientation & stability tracking  
-  Mouse movement and activity monitoring  
-  Tab switch detection (Page visibility tracking)  
-  Dynamic behavioral attention scoring  
-  WebSocket-based real-time data streaming  
-  Session logging and backend storage  
-  Scalable architecture (Frontend + Backend)

## 🧠 How It Works

1. Webcam captures live video stream.  
2. OpenCV detects facial presence and movement.  
3. Eye blink rate is calculated to monitor engagement or fatigue.  
4. Head orientation and stability are analyzed.  
5. Mouse movement patterns are tracked to measure interaction level.  
6. Tab switches and page visibility changes are detected.  
7. A dynamic attention score is calculated based on combined behavioral metrics.  
8. Data is streamed to backend via WebSocket.  
9. Session analytics are generated and stored.
